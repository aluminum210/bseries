# BBag add-on.
An add-on for World of WarCraft: Wrath of the Lich King game.
The add-on incorporates a minimalist inventory manager into the game client.
Inspired by ArkInventory and The Elder Scrolls: Oblivion inventory system.
## Features.
1. Higher performance than ArkInventory. 
2. Search items by name or by category.
3. Meaningful presets: no set-up and user configuration requried.
